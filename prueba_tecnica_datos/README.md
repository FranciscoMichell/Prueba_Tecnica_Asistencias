\# Prueba Técnica - Análisis de Datos



\## Objetivo

Limpieza y análisis de datos para identificar participantes únicos y su nivel de asistencia.



\## Proceso realizado

\- Limpieza de datos y manejo de valores nulos

\- Estandarización de nombres (Title Case sin acentos)

\- Consolidación de registros por participante (Email)

\- Cálculo de asistencia total por participante

\- Identificación de participantes con menor desempeño



\## Resultado

Se identificaron los participantes con menor desempeño como aquellos con menor número de asistencias registradas.



\## Archivos

\- `dataset\_final.csv`: dataset limpio y consolidado

\- `peor\_desempeno.csv`: participantes con menor asistencia
